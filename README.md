# 🏋️ Ruden Fitness

**Elite Personal Training by Rudrendra Shrestha**

Premium personal trainer website built with Next.js 14, TypeScript, TailwindCSS, Framer Motion, Firebase, and Sanity CMS.

## Brand
- **Coach:** Rudrendra Shrestha
- **Instagram:** [@ruden_s](https://instagram.com/ruden_s)
- **Profession:** Personal Trainer & Men's Physique Athlete

## Tech Stack
- Next.js 14 (App Router)
- TypeScript
- TailwindCSS
- Framer Motion
- Firebase (Auth + Database)
- Sanity CMS (Blog)

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).
